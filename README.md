# bcryptjs-webcrypto
This is a fork of bcryptjs.  
This removes Node.js import.  
So doesn't work it on Node.js <23.  

Usage and other information are available in the original.  
Please see [bcrypt.js](https://github.com/dcodeIO/bcrypt.js).

## Install
```bash
$ bun install bcryptjs-webcrypto
```
